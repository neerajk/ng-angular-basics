** Angular Architecute Overview **
There are 5 building blocks in angular: 
1. Modules: Building blocks that contains routes, components,  services and more...
So, we can have multiple modules, single module and modules that depends on other modules.
2. Components: Contains a template, data, logic, and formating part of DOM tree.
3. Directives: Attached behaviour, extend and transform a particualr element and its children
4. Routing: Renders a component based on url state, drive navigation.
5. Services: Data layer, logic that is not component related, such as API requests.
